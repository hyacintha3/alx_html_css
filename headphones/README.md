# Headphones — ALX HTML/CSS Project

This project reproduces the Headphones landing page (Header, Hero, "What we do", "Our results", Contact, Footer).
Mobile breakpoint: 480px (switches to mobile layout).

Files:
- 0-index.html / 0-styles.css — Header & hero
- 1-index.html / 1-styles.css — What we do
- 2-index.html / 2-styles.css — Our results (with CSS pentagons)
- 3-index.html / 3-styles.css — Contact form
- 4-index.html / 4-styles.css — Footer
- 6-index.html / 6-styles.css — Pentagons (alternate file demonstrating shapes)
- 7-index.html / 7-styles.css — Animations
- 8-index.html / 8-styles.css / 8-script.js — Hamburger menu (JS allowed for this task)

How to preview:
1. Open the `headphones` folder in VS Code.
2. Install Live Server extension.
3. Right-click any `*-index.html` file and choose "Open with Live Server".
